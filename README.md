# JavaScript Calculator 🧮

A simple and user-friendly calculator application built using HTML, CSS, and JavaScript.

This calculator performs basic arithmetic operations and provides a simple interface for users to calculate values easily.

## 🚀 Features

- Addition operation
- Subtraction operation
- Multiplication operation
- Division operation
- Clear/reset functionality
- Simple and clean user interface
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
